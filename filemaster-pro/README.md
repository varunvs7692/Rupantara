# FileMaster Pro

A modern web application for file compression, PDF conversion, and advanced image tools.

## Features
- File Compression (PDF, DOCX, PPTX, TXT, ZIP)
- File to PDF Converter (DOCX, PPTX, TXT, Images)
- Image Compression & AI Enhancement
- Clean, modern UI (React.js + Tailwind CSS)
- Node.js + Express.js backend
